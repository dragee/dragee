MultiDrag.Draggable

MultiDrag.Draggable(element[, options])

Options

parent HTMLElement який визначає декартову систему координат для MultiDrag.Draggable, його верхній лівий кут буде визначений як точка new Point (0,0).

bound дозволяє задати функцію, що описує граничні обмеження для переміщення елементу. В модулі bound.js ви можете знайти декілька корисних функцій для побудови граничних обмежень.

position - координата за замовчуванням. Якщо position не задана, координата вираховується автоматично згідно з розміщенням в батьківсьмому контейнері.

isContentBoxSize - визначає чи повинні ми враховувати padding при визначенні розміру блоку. Callbacks: onEnd, onMove

Значення по замовчуванню:
parent: parent = MultiDrag.util.getDafaultParent(element),
bound: function(){ return MultiDrag.boundFactory(MultiDrag.boundType.element)(parent, parent); },
onEnd: function(){},
onMove: function(){}

Methods

getSize()
getCenter()
getRectangle()
move(point, time, isFix, isSilent) - переміщує елемент в точку point за час time.
setPosition(point)

Properties

enable = [true|false]

MultiDrag.Target

MultiDrag.Target

"Кошик" в який можна додати MultiDrag.Draggable та сортувати їх. Самостійне використанняне не передбачене, використовується спільно з MultiDrag.TargetManager.

Target(element, draggables[, options]) Конструктор включає 3 параметра :
element - HTMLElement "кошика" який визначає розміри та позицію

Options:

timeEnd - час переміщення MultiDrag.Draggable в цільову точку.
timeExcange - час за який MultiDrag.Draggables обмінюються своїми позиціями.
parent - HTMLElement який визначає декартову систему координат що буде використовуватися
sorting - функція яка визначає, яким чином мають впорядковуватися блоки в "кошику".
B модулі position.js ви можете знайти декілька корисних функцій(sortingFactory(type))
positioning - функція яка визначає, яким чином мають розташовуватися блоки в "кошику".
В модулі position.js ви можете знайти декілька корисних функцій(positionFactory(type))

Callbacks: onAdd - функція, яка викликається при додаванні блоку в "кошик". onRemove - функція, яка викликається при видаленні блоку з "кошика".

Defaults:

timeEnd: 200,
timeExcange: 400,
parent: parent,
sorting: MultiDrag.sortingFactory(MultiDrag.positionType.floatLeft)(80, mathPoint.getLength({x: 1, y:4, IsTransformationSpace:true})),
positioning: MultiDrag.positionFactory(MultiDrag.positionType.floatLeft)(MultiDrag.util.bind(this.getRectangle,this), {isRemove: true})

Methods:

getSortedDraggables() - метод який повертає впорядкований список MultiDrag.Draggable
reset() - відновлення початкових позицій блоків(видалення з "кошика")

MultiDrag.TargetManager

Менеджер роботи групи "кошиків".

TargetManager(element, draggables, targets, options)
element - HTMLElement який визначає робочу область
draggables - масив MultiDrag.Draggable, з якими працює менеджер
targets - масив MultiDrag.Target, з якими працює менеджер

Options:

onChange функція-слухач яка викликається при переміщення елемента (з "кошика" чи в "кошик").
timeEnd час переміщення MultiDrag.Draggable з однієї точки в іншу.

Defaults:
timeEnd: 400,

Methods:

reset() - відновлення початкових позицій елементів (видалення з "кошиків")

Properties:

positions - позиції елементів в "кошиках", зручно використовувати для збереження та відновлення стану

Factory:

MultiDrag.targetManagerFactory(element, draggableElements, targetElements, options). options = { draggable: {...}, target: {...}, manager: {...} }

MultiDrag.List

MultiDrag.List(draggables[, options])

Options:

timeEnd - час переміщення MultiDrag.Draggable в кінцеву точку.
timeExcange - час за який MultiDrag.Draggables обмінюються своїми позиціями.
getLength - функція яка визначає яким чином буде визначатися відстань між 2 точками в декартовій системі координат(метрика простору), необхідна для визначення "влучення в ціль". Модуль mathPoint містить фабрику getLength(options) для побудови базових метрик.
radius - радіус при попаданні блока в який ми вважаємо, що він попав в ціль.
Callbacks: onChange

Defaults:

timeEnd: 200,
timeExcange: 400,
radius: 30,
getLength: mathPoint.getLength()

Methods:

getSortedDraggables() - метод який повертає впорядкований список блоків

Properties:

positions - позиції блоків, зручно використовувати для збереження та відновлення стану

Factory:

Для спрощення ініціалізації списка існує фабика MultiDrag.listFactory(parentElement, elements, options).
options = { draggable : {...} , list : {...} }

MultiDrag.Spider

"Сітка" - використовується для задання вектору вибору, вона складається з променів по яких рухаються повзунки. Позиціонувати елементи не потрібно їх позиції задасть MultiDrag.Spider при ініціалізації.

Spider(area, elements, options)
area - HTMLElement який визначає робочу область
elements - масив HTMLElement-ів, які будуть задавати вектори
targets - масив MultiDrag.Target, з якими працює менеджер

Options:

angle початковий кут від якого будуть відкладатися кути інших "променів"
dAngle кут між 2 "променями"
center центр від якого відходять "промені"
startRadius нижнє обмеження повзунка
endRadius верхнє обмеження повзунка
lineWidth, strokeStyle, fillStyle стилізація "сітки"

Defaults:
angle: 0,
dAngle: 2 * Math.PI / elements.length,
center: areaRectangle.getCenter(),
startRadius: 50,
endRadius: mathPoint.createRectangleFromElement(area, area).getMinSide() / 2,
lineWidth: 2,
strokeStyle: "#ff5577",
fillStyle: "rgba(150,255,50,0.8)"

mathPoint

Methods:

getAVG([point1, point2, point3, ...]) - повертає середнє арифметичне вхіднич точок
getLength(options) - фабрика побудови метрики (функції для визначення відстані між 2 точками), options={x, y, isTransformationSpace} задання властивостей x, y додає вісі в метрику визначення відстаней, якщо прапор isTransformationSpace заданий true, тоді відбувається трансформація простору, а коефіціенти x,y задають розтягнення стиснення координат, опція трасформації необхідна для виділення пріоритетної вісі, яка є більш важливою при виявленні наближених точок.
indexOfNearPoint(arr, val, radius, getLength) - повертає індекс найближчої точки з масива arr до заданої val, пошук точок обмежується радіусом radius, а спосіб визначення відстанні передається як функція getLength.
bound(min, max, val) - приводить число до входження в заданий інтервал
boundPoint(min, max, val) - приводить точку до входження в прямокутну область
directCrossing(L1P1, L1P2, L2P1, L2P2) - повертає точку пересічення 2 прямих
boundOnLine(LP1, LP2, P) - переносить задану точку P на пряму LP1, LP2 шляхом відкладання перпендикуляра на пряму
getPointInLine(LP1, LP2, percent) - знаходить точку на відрізку яка знаходиться на відстані percent*(LP2-LP1) від першої точки
getPointInLineByLenght(LP1, LP2, lenght) - знаходить точку на відрізку, яка знаходиться на відстані lenght від першої точки
createRectangleFromElement(element, parent, isContentBoxSize, isConsiderTranslate) - створює прямокутник на основі елемента, центр декартової системи задає верхній лівий кут parent-а, isContentBoxSize - прапор який включає в розміри прямокутника padding та border, isConsiderTranslate - прапор врахування css властивостей -webkit-translate при знаходженні позиції
getSizeOfElement(element, isContentBoxSize)
getOffset(element, parent, isConsiderTranslate) - знаходить позицію елемента
getPointFromRadialSystem(angle, length, center) - переводить в декартову ситсему, точку заданою полярною системою
addPointToBoundPoints(boundpoints, point, isRight)

position

Модуль складається з 2 фабрик.

MultiDrag.sortingFactory дозволяє створити функцію, що буде сортувати блоки.

MultiDrag.positionFactory дозволяє створити функцію, що буде контролювати розміщення блоків.

Поведінка залежить від обраної стратегії.

MultiDrag.positionType.notCrossing 

MultiDrag.sortingFactory(MultiDrag.positionType.notCrossing)()
MultiDrag.positionFactory(MultiDrag.positionType.notCrossing)(rectangle, options)
при позиціонуванні з данною стратегією блоки зміщуються таким чином щоб уникнути перетинання

MultiDrag.positionType.floatLeft 
MultiDrag.sortingFactory(MultiDrag.positionType.floatLeft)(radius, getLength, options)
MultiDrag.positionType.floatRight 
MultiDrag.sortingFactory(MultiDrag.positionType.floatRight)(radius, getLength, options)
